## Installation & Setup

First, make sure your project is using **React 18+** and **Tailwind CSS**.

**Install the CLI globally:**
```bash
npm install -g shadcn-ui@latest
```

**Initialize ShadCN UI in your project:**
```bash
npx shadcn-ui@latest init
```
Or, for specific environments:
```bash
npx shadcn-ui@latest init --app ./src --ts
```

**Follow the prompts:**  
You'll select a target folder, theme preference, and a few config options. This process scaffolds the component structure and sets up your project for incremental adoption.

---

## Configuration Essentials

After initialization, you'll have:

- **`components.json`:** Configuration file that defines your component setup
- **`lib/utils.ts`:** Utility functions including the `cn()` function for class merging
- **Tailwind Config:** Updated to include ShadCN UI's color variables and component paths

**Key configuration options:**
```json
{
  "style": "default",
  "rsc": true,
  "tsx": true,
  "tailwind": {
    "config": "tailwind.config.js",
    "css": "app/globals.css"
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils"
  }
}
```

## Best Practices

- **Incremental Adoption:** Add components one at a time‚Äîno need to import everything upfront
- **Custom Styling:** Override Tailwind classes for branding without deep-diving into library internals
- **Accessibility First:** Components ship with ARIA attributes and keyboard support‚Äîdon't remove them!
- **Component Updates:** Keep your components up to date with `npx shadcn-ui@latest add <component>`
- **Performance:** Only include components you use. No bundle bloat!
- **Version Control:** Commit the generated components to your repo for full control

## Example: Adding a Button Component

**Step 1: Generate the Button**
```bash
npx shadcn-ui@latest add button
```

This creates `components/ui/button.tsx` with all the variants and styling.

![Button component code examples in VSCode](generated_image:4)

**Step 2: Use the Button Component**
```jsx
import { Button } from "@/components/ui/button";

export default function HomePage() {
  return (
    <section className="p-8">
      <h1 className="mb-4 text-2xl font-bold">Welcome to My Portfolio!</h1>
      <div className="flex gap-4">
        <Button variant="default" size="lg">
          Primary Action
        </Button>
        <Button variant="outline" size="lg">
          Secondary Action
        </Button>
      </div>
    </section>
  );
}
```

**Available Variants:**
```jsx
<Button variant="default">Default</Button>
<Button variant="destructive">Destructive</Button>
<Button variant="outline">Outline</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="link">Link</Button>
```

**Available Sizes:**
```jsx
<Button size="default">Default</Button>
<Button size="sm">Small</Button>
<Button size="lg">Large</Button>
<Button size="icon">Icon</Button>
```

## Customizing Components

ShadCN UI components are just regular React components living in your codebase. You can modify them directly or extend them as needed.

**Custom Button Example:**
```jsx
import { Button } from "@/components/ui/button";
import { cn } from "@/lib/utils";

export function GradientButton({ 
  children, 
  className, 
  ...props 
}) {
  return (
    <Button 
      className={cn(
        "bg-gradient-to-r from-indigo-500 to-cyan-500 text-white hover:from-indigo-600 hover:to-cyan-600",
        className
      )}
      {...props}
    >
      {children}
    </Button>
  );
}
```

**Adding Icons:**
```jsx
import { Button } from "@/components/ui/button";
import { Download } from "lucide-react";

export function DownloadButton() {
  return (
    <Button>
      <Download className="mr-2 h-4 w-4" />
      Download CV
    </Button>
  );
}
```

## Theme Support

ShadCN UI provides excellent dark/light mode support out of the box.

**Setting up the theme provider:**
```jsx
// app/layout.tsx
import { ThemeProvider } from "@/components/theme-provider";

export default function RootLayout({ children }) {
  return (
    <html lang="en" suppressHydrationWarning>
      <body>
        <ThemeProvider
          attribute="class"
          defaultTheme="system"
          enableSystem
          disableTransitionOnChange
        >
          {children}
        </ThemeProvider>
      </body>
    </html>
  );
}
```

**Theme toggle component:**
```jsx
// components/theme-toggle.tsx
import { Moon, Sun } from "lucide-react";
import { useTheme } from "next-themes";
import { Button } from "@/components/ui/button";

export function ThemeToggle() {
  const { setTheme, theme } = useTheme();

  return (
    <Button
      variant="ghost"
      size="icon"
      onClick={() => setTheme(theme === "light" ? "dark" : "light")}
    >
      <Sun className="h-[1.5rem] w-[1.3rem] dark:hidden" />
      <Moon className="hidden h-5 w-5 dark:block" />
      <span className="sr-only">Toggle theme</span>
    </Button>
  );
}
```

## More Component Examples

**Adding a Card component:**
```bash
npx shadcn-ui@latest add card
```

```jsx
import {
  Card,
  CardContent,
  CardDescription,
  CardHeader,
  CardTitle,
} from "@/components/ui/card";

export function ProjectCard({ title, description, tech }) {
  return (
    <Card className="w-full max-w-sm">
      <CardHeader>
        <CardTitle>{title}</CardTitle>
        <CardDescription>{description}</CardDescription>
      </CardHeader>
      <CardContent>
        <div className="flex flex-wrap gap-2">
          {tech.map((item) => (
            <span
              key={item}
              className="px-2 py-1 text-xs bg-secondary rounded-md"
            >
              {item}
            </span>
          ))}
        </div>
      </CardContent>
    </Card>
  );
}
```
